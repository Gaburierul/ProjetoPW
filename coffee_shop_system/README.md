# Coffee Shop System (Versão Estudante)
Sistema PHP simples para gerenciar uma cafeteria — pronto para rodar localmente.
Baseado no projeto de sistema de aluguel de carros como modelo.

## Requisitos
- Apache/Nginx + PHP 8.x com PDO (pdo_mysql)
- MariaDB / MySQL

## Instalação rápida
1. Ajuste as credenciais no arquivo `config/db.php`.
2. Importe `database.sql` no seu servidor de banco de dados.
3. Coloque a pasta `public/` no diretório público do servidor (ex.: `htdocs/coffee_shop_system/public`).
4. Acesse `http://localhost/caminho/para/public/login.php`.

Usuário admin inicial:
- usuário: `admin`
- senha: `admin123`
